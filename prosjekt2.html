Grunnleggende web

Dette temaet lærte jeg det mest grunnleggende om koding, hvor jeg blant annet lærte hva HTML og CSS er. HTML brukes til å lage strukturen på en nettside, mens CSS brukes for å styre utseendet og designet. Jeg fikk også forståelse for mappestruktur, altså hvordan man organiserer filer og mapper for en nettside. Dette var imidlertid et vanskelig prosjekt for meg, spesielt å sette opp og forstå hvordan man får nettsiden på nettet. Jeg lærte blant annet hvordan man bruker FileZilla for å laste opp filene til one.com. Jeg synes det var vanskelig å forstå koding til å begynne med, da jeg tenkte alt for komplisert.
